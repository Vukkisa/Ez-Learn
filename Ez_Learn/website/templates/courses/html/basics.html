1. What is HTML? üöÄ

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses tags to structure content.

<!DOCTYPE html> <html> <head> <title>My First Web Page</title> </head> <body> <h1>Hello, World!</h1> </body> </html>

2. Basic HTML Structure üèóÔ∏è

Every HTML document has a basic structure:

<!DOCTYPE html> <!-- Document type --> <html lang="en"> <!-- Root element --> <head> <!-- Meta information --> <meta charset="UTF-8"> <title>Page Title</title> </head> <body> <!-- Visible content --> <!-- Your content here --> </body> </html>

3. Common HTML Tags

Learn the most essential HTML tags:

<h1>Main Heading</h1> <!-- Largest heading --> <h2>Sub Heading</h2> <!-- Smaller heading --> <p>This is a paragraph.</p> <!-- Paragraph text --> <strong>Bold text</strong> <!-- Bold/strong text --> <em>Italic text</em> <!-- Italic/emphasized text --> <br> <!-- Line break --> <a href="https://example.com">Link</a> <!-- Hyperlink -->

4. Lists - Organized Content

Create ordered and unordered lists:

<!-- Unordered List (bullets) --> <ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul> <!-- Ordered List (numbers) --> <ol> <li>First step</li> <li>Second step</li> <li>Third step</li> </ol>

5. Images and Links

Add images and create links to other pages:

<!-- Adding an image --> <img src="photo.jpg" alt="Description" width="300" height="200"> <!-- Creating links --> <a href="https://www.google.com">Visit Google</a> <a href="about.html">About Page</a> <a href="#section1">Jump to Section</a> <!-- Email link --> <a href="mailto:contact@example.com">Send Email</a>
