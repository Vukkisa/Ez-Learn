1. CSS Animations and Keyframes

Create smooth, professional animations using CSS keyframes and transitions.

/* Keyframe Animation */ @keyframes slideInFromLeft { 0% { transform: translateX(-100%); opacity: 0; } 100% { transform: translateX(0); opacity: 1; } } @keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.1); } } /* Animation Classes */ .slide-in { animation: slideInFromLeft 1s ease-out forwards; } .pulse-effect { animation: pulse 2s infinite; } /* Advanced Transition */ .button { background: #1572b6; color: white; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; /* Transition */ transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1); /* Transform origin */ transform-origin: center; } .button:hover { background: #0d5a99; transform: translateY(-2px) scale(1.05); box-shadow: 0 8px 25px rgba(21, 114, 182, 0.3); } .button:active { transform: translateY(0) scale(0.98); transition: all 0.1s ease; }

2. CSS Variables and Custom Properties

Master CSS custom properties for maintainable and dynamic styling.

/* CSS Custom Properties (Variables) */ :root { /* Color Scheme */ --primary-color: #1572b6; --secondary-color: #28a745; --accent-color: #ffc107; --danger-color: #dc3545; /* Typography */ --font-family-primary: 'Inter', sans-serif; --font-size-base: 16px; --line-height-base: 1.6; /* Spacing */ --spacing-xs: 0.25rem; --spacing-sm: 0.5rem; --spacing-md: 1rem; --spacing-lg: 1.5rem; --spacing-xl: 3rem; /* Shadows */ --shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.12); --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.15); --shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.2); /* Border Radius */ --radius-sm: 4px; --radius-md: 8px; --radius-lg: 12px; } /* Using Variables */ .button { background: var(--primary-color); color: white; padding: var(--spacing-sm) var(--spacing-md); border-radius: var(--radius-md); font-family: var(--font-family-primary); font-size: var(--font-size-base); line-height: var(--line-height-base); box-shadow: var(--shadow-md); } /* Dynamic Theme Switching */ [data-theme="dark"] { --primary-color: #66b3ff; --secondary-color: #4ade80; --bg-color: #1a1a1a; --text-color: #ffffff; } [data-theme="light"] { --primary-color: #1572b6; --secondary-color: #28a745; --bg-color: #ffffff; --text-color: #333333; }

3. Advanced Selectors and Pseudo-elements

Master complex selectors and create dynamic content with pseudo-elements.

/* Advanced Selectors */ /* Attribute Selectors */ input[type="text"] { border: 2px solid #ccc; } input[data-validation="error"] { border-color: var(--danger-color); box-shadow: 0 0 5px rgba(220, 53, 69, 0.3); } /* Combinators */ .nav > li { /* Direct child */ margin: 0 10px; } .content h2 + p { /* Adjacent sibling */ margin-top: 0; } .article h2 ~ p { /* General sibling */ color: #666; } /* Pseudo-classes */ .button:nth-child(even) { /* Even elements */ background: var(--secondary-color); } .tr:hover .delete-btn { /* Show on parent hover */ opacity: 1; visibility: visible; } .delete-btn { opacity: 0; visibility: hidden; transition: all 0.3s ease; } /* Pseudo-elements */ .quote::before { content: """; font-size: 4rem; color: var(--primary-color); position: absolute; top: -10px; left: -20px; } .quote::after { content: """; font-size: 4rem; color: var(--primary-color); position: absolute; bottom: -30px; right: -20px; } /* Custom Scrollbar */ ::-webkit-scrollbar { width: 12px; } ::-webkit-scrollbar-track { background: #f1f1f1; border-radius: var(--radius-sm); } ::-webkit-scrollbar-thumb { background: var(--primary-color); border-radius: var(--radius-sm); } ::-webkit-scrollbar-thumb:hover { background: #0d5a99; }
