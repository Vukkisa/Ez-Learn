1. What is CSS?

CSS (Cascading Style Sheets) controls the visual appearance of HTML elements. It's what makes websites beautiful!

/* This is a CSS comment */ /* Change text color */ p { color: blue; } /* Change background */ body { background-color: lightgray; }

2. CSS Syntax

CSS follows a simple syntax rule:

selector { property: value; property: value; } /* Example */ h1 { color: red; font-size: 24px; text-align: center; }

3. Colors and Typography

Control colors and text appearance:

/* Text colors */ .blue-text { color: blue; } .green-text { color: #00ff00; } .red-text { color: rgb(255, 0, 0); } /* Font styling */ .bold-text { font-weight: bold; } .italic-text { font-style: italic; } .large-text { font-size: 20px; } .small-text { font-size: 12px; }

4. Backgrounds and Borders

Style backgrounds and add borders to elements:

/* Background colors */ .background-blue { background-color: lightblue; } .background-red { background-color: #ffcccc; } /* Borders */ .border-solid { border: 2px solid black; } .border-dashed { border: 1px dashed red; } .border-rounded { border-radius: 10px; }

5. Basic Layout - Box Model

Understand how elements are sized and spaced:

.box { width: 200px; /* Width of the content */ height: 100px; /* Height of the content */ padding: 20px; /* Space inside the element */ margin: 10px; /* Space outside the element */ border: 2px solid black; /* Border around the element */ } /* Center text */ .center-text { text-align: center; } .center-div { margin-left: auto; margin-right: auto; }
