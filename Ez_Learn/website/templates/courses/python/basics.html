5. Lists - Your First Data Structure üìã

Lists are used to store multiple items in a single variable. Think of them as ordered containers!

# Creating and using lists fruits = ["apple", "banana", "orange"] print(fruits[0]) # Access first item (indexing starts at 0) print(fruits[-1]) # Access last item fruits.append("grape") # Add new item to the end fruits.insert(1, "kiwi") # Insert at specific position print(f"Total fruits: {len(fruits)}") print(fruits)

6. Loops - Repeat Actions üîÑ

Loops help you repeat code multiple times. Two main types: for loops and while loops.

# For loop - iterate through a list fruits = ["apple", "banana", "orange"] for fruit in fruits: print(f"I like {fruit}") # While loop - repeat while condition is true count = 0 while count < 3: print(f"Count: {count}") count += 1 # Same as count = count + 1

8. String Methods - Text Processing üìù

Python has powerful built-in methods for working with text (strings).

text = " Hello Python World " # String methods print(f"Original: '{text}'") print(f"Stripped: '{text.strip()}'") print(f"Upper: '{text.upper()}'") print(f"Lower: '{text.lower()}'") print(f"Replace: '{text.replace('Python', 'Programming')}'") print(f"Split: {text.split()}") # Splits into a list of words
