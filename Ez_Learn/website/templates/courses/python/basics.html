1. What is Python?

Python is a high-level, interpreted programming language with simple syntax and powerful capabilities. It's perfect for beginners!

# Your first Python program print("Hello, World!")

2. Variables and Data Types

In Python, you can store different types of data in variables:

# Different data types name = "Alice" # String age = 25 # Integer height = 5.6 # Float is_student = True # Boolean print(f"Name: {name}") print(f"Age: {age}")

3. Basic Operations

Python supports all basic mathematical operations:

a = 10 b = 3 # Basic operations print(a + b) # Addition print(a - b) # Subtraction print(a * b) # Multiplication print(a / b) # Division print(a % b) # Modulus (remainder)

4. Conditional Statements

Use if/else statements to make decisions in your code:

score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" else: grade = "F" print(f"Your grade: {grade}")

5. Lists - Your First Data Structure

Lists are used to store multiple items in a single variable:

# Creating and using lists fruits = ["apple", "banana", "orange"] print(fruits[0]) # Access first item fruits.append("grape") # Add new item print(f"Total fruits: {len(fruits)}")
