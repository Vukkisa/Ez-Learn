Inheritance

Python is an Object-Oriented Programming language and one of the features of Object-Oriented Programming is Inheritance. Inheritance is the ability of one class to inherit another class. Inheritance provides reusability of code and allows us to create complex and real-world-like relationships among objects.

Nomenclature of Python Inheritance
The class which got inherited is called a parent class or superclass or base class. The class which inherited the parent class is called a child class or subclass or derived class or extended class.

Polymorphism is built-in functions

The built-in function len() calculates the length of an object depending upon its type. If an object is a string, it returns the count of characters, and If an object is a list, it returns the count of items in a list.

The len() method treats an object as per its class type.
